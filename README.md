# Nattanon Sripauraya — Portfolio

Personal portfolio website for **Nattanon Sripauraya** (ณัฐฐ์ฐนนท์ ศรีเปารยะ), Full Stack Developer specializing in Web, Android, and DevOps.

🌐 **Live site:** [https://nattanonsri.github.io/portfolio/](https://nattanonsri.github.io/portfolio/)

## Features

- Thai / English language switch
- Dark mode
- Responsive design
- Auto-deploy to GitHub Pages

## Tech Stack

- React 19 + TypeScript
- Vite 6
- Tailwind CSS 4

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
npm run preview
```

## Deploy

Pushing to `main` triggers GitHub Actions and deploys to GitHub Pages automatically.
